1. create dir
2. create index.html
3. copy and paste: 
        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="utf-8">
                <title>My first three.js app</title>
                <style>
                    body { margin: 0; }
                </style>
            </head>
            <body>
                <script type="module" src="/main.js"></script>
            </body>
        </html>
4. create main.js
5. copy and paste:
        import * as THREE from 'three';
6. open terminal 
7. npm install --save three
8. npm install --save-dev vite
9. to run application run:
        npx vite

if it does not work refer to: https://threejs.org/docs/#manual/en/introduction/Installation
